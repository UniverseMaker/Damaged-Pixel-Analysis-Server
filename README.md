# Damaged-Pixel-Analysis-Server
Damaged Pixel Analysis Software based on PHP Environment. This software requires pre-processing standalone software. 손상 픽셀 분석 시스템이며 본 시스템을 사용하기 위해서는 전처리 소프트웨어가 추가적으로 필요합니다.

# Demo
